# 🔐 Authentication System (Styled)

This project is a responsive and visually enhanced authentication system built using HTML, Bootstrap 5, and custom CSS.

It includes all essential authentication pages with modern UI/UX design and responsive layout.

---

## 📄 Pages Included

1. **login.html** – Login Page  
2. **register.html** – Registration Page  
3. **forgot-password.html** – Forgot Password Page  
4. **reset-password.html** – Reset Password Page  
5. **dashboard.html** – User Dashboard  

---

##  Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Bootstrap 5 integration
- Custom CSS styling (styles.css)
- Google Fonts (Poppins)
- Card-based UI layout
- Form validation using HTML5
- Password visibility toggle 
- Clean and professional UI
- Smooth hover effects and transitions

---

##  Technologies Used

- HTML5  
- Bootstrap 5  
- CSS3
